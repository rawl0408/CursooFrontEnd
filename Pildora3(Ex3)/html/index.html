Todas las secciones dentro de article tendrán este formato.

Todas las "p" dentro de seccion y que estén dentro de article, tendrán este formato.

Todas las "p" en el body, tendrán este formato.

Todas los "H2" que sean hijos de DIV's, tendrán este formato.

- El "border" es una linea que se pinta alrededor del item donde lo pongas.

- El "padding" es el espacio que hay entre el border y el contenido interior, ya sea imagen, texto...

- El "margin" es el espacio que se añade entre el border hacia fuera
